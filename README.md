# deathofaheart.com

Clean, professional portfolio for Concert & Portrait Photography.
Red/Black theme.

## Features
- Landing page with fade-out intro animation.
- Navigation to Concerts, Portraits, Contact.
- Responsive Gallery Layouts (Placeholders active).
- Framer Motion animations.

## Setup
1. `npm install`
2. `npm run dev` to start local server.
3. `npm run build` to build for production.

## Stack
- Vite + React
- Framer Motion
- React Router DOM
